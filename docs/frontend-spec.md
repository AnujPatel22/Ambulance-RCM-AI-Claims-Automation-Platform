# Frontend Spec

The frontend is a Vite React app with TypeScript and plain CSS.

## Pages

- Dashboard: metric overview and simulated operational improvement.
- Ambient Documentation: select or paste a transcript and extract PCR fields.
- Claims Queue: review claims and run validation.
- Denial Management: review denied claims and generate appeals.
- Rules Explorer: search synthetic payer rules.

## UX Principles

- Use real app screens instead of a marketing landing page.
- Keep controls obvious and workflows short.
- Show loading, error, and empty states.
- Keep visual styling professional and calm for an operational healthcare workflow.
