# Research Repos

The project is inspired by public patterns in healthcare RCM, denial management, payer-policy retrieval, and AI agent portfolio projects. No code is copied from other repositories.

Nearby inspiration themes:

- Claim denial analysis and missing evidence detection.
- Synthetic denial corpora and PHI-safety ideas.
- Healthcare claims RAG and policy retrieval.
- Billing automation workflows.
- EMS PCR documentation concepts.

This implementation uses only synthetic data and simplified ambulance billing logic suitable for a GitHub portfolio MVP.
